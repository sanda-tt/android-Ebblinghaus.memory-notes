# android-Ebblinghaus.memory-notes
基于艾宾浩斯遗忘曲线的智能背诵提醒系统设计与应用（安卓app）

## 应用介绍
### 应用原理
基于艾宾浩斯理论，系统根据遗忘算法，自动在日历中生成相对日期的艾宾浩斯事件，提醒背诵计划，

### 解决痛点
1. 记忆不牢固，容易遗忘
2. 普通提醒软件采用软件内提醒策略需要多项权限，包括通知，后台运行和自启动等，容易收不到提醒且消耗电量。此app采取写入手机日历的策略理论上杜绝了上述痛点。
### 功能特点
1. 多情况自定义录入，支持自定义记忆内容
2. 结构化组织，结合研究算法，生成最佳方案自动生成在1、2、6、14、30天的背诵提醒
3. 超模块更新和删除，更新清晰显示艾宾浩斯实践，支持批量删除艾宾浩斯事件
## 使用说明
### 通用
1. 添加事件备注（如“英语课本第一单元”或“数学公式一”等）
2. 点击添加艾宾浩斯事件即可
